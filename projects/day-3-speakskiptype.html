SpeakSkipType

Speech-to-Text Directly at Your Cursor - 100% Offline

A developer tool that converts your voice to text and types it wherever your cursor is. Dual speech recognition engines (Vosk for speed, Whisper for accuracy), automatic filler word removal, and Claude Code integration. Works completely offline.

The Problem

Voice-to-text tools have significant limitations:

  • Require constant internet connection
  • Privacy concerns - audio sent to cloud
  • Can't type directly where you're working
  • No integration with developer tools
  • Filled with "um", "uh", and filler words

The Solution

SpeakSkipType brings voice input to developers:

  • 100% offline - no data leaves your machine
  • Types directly at cursor position
  • Automatic filler word removal
  • Voice commands for punctuation
  • Claude Code integration for AI commands

Key Features

Dual Engine

Vosk for lightning-fast transcription, Whisper for maximum accuracy. Switch based on your needs.

100% Offline

All processing happens locally. Your voice data never touches the internet. Complete privacy.

Cursor Position

Text appears exactly where your cursor is. Works in any application - IDEs, browsers, terminals.

Filler Removal

Automatically removes "um", "uh", "like", and other filler words. Clean, professional output.

Voice Commands

Say "period", "comma", "new line" for punctuation. Natural dictation without breaking flow.

306+ Tests

Comprehensive test suite ensures reliability. Actively maintained with MIT license.

How It Works

1

Press Alt+R to Record

Start recording your voice with a simple keyboard shortcut:

Alt + R # Start recording
2

Speak Naturally

Talk as you normally would. Use voice commands for punctuation:

"Hello world period This is a test comma right question mark"
3

Press Alt+S to Transcribe

Stop recording and automatically paste the transcription:

Alt + S # Stop and transcribe
4

Text Appears at Cursor

Clean, punctuated text typed where your cursor was:

Hello world. This is a test, right?

Quick Start

1

Clone & Install

git clone https://github.com/DandaAkhilReddy/Speakskiptype.git
cd Speakskiptype
pip install -r requirements.txt
2

Download Vosk Model

python -c "import vosk; vosk.Model(lang='en-us')"
3

Run the Application

python main.py
