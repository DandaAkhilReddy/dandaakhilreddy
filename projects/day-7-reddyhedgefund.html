AI Hedge Fund

18 Legendary Investor AI Agents Analyzing Stocks with Fundamentals, Technicals, Sentiment & Valuation

An AI-powered hedge fund simulation where 18 specialized agents — modeled after Warren Buffett, Charlie Munger, Aswath Damodaran, Cathie Wood, Rakesh Jhunjhunwala, and more — collaboratively analyze stocks using fundamental analysis, technical indicators, sentiment scoring, and valuation models. Built with Python and LangGraph for multi-agent orchestration. Educational purposes only.

The Problem

Traditional stock analysis has significant limitations:

  • Manual analysis is slow — hours per stock, per framework
  • Individual bias dominates decision-making
  • Hard to apply multiple investing philosophies simultaneously
  • Fundamentals, technicals, and sentiment analyzed in silos
  • No systematic way to aggregate diverse expert opinions

The Solution

AI agents embodying legendary investors, working as a team:

  • 18 specialized agents with distinct investing philosophies
  • Parallel analysis across fundamentals, technicals, sentiment, valuation
  • Risk manager aggregates signals and enforces position limits
  • Portfolio manager generates final buy/sell/hold decisions
  • LangGraph orchestrates the entire multi-agent workflow

Core Capabilities

Legendary Investors

12 investor agents modeled after Buffett, Munger, Damodaran, Cathie Wood, Rakesh Jhunjhunwala, Soros, and more — each with a unique investing philosophy.

Valuation Models

DCF analysis, owner earnings, intrinsic value calculations, and multi-framework valuation — applied automatically to every stock analyzed.

Sentiment Analysis

Analyzes market sentiment from news, insider trading patterns, and social signals to gauge market psychology around each stock.

Technical Analysis

Moving averages, RSI, MACD, Bollinger Bands, and volume analysis — computed programmatically and interpreted by specialist agents.

Risk Management

Aggregates all agent signals, enforces position sizing limits, calculates portfolio-level risk metrics, and prevents overconcentration.

Portfolio Optimization

Final portfolio manager synthesizes all analysis into actionable trading signals — buy, sell, or hold with conviction scores and position sizes.

Multi-Agent Architecture

Data flows through a LangGraph-orchestrated pipeline where each agent contributes specialized analysis before the risk manager and portfolio manager make final decisions.

Setup Guide

1

Clone the Repository

Get the source code from GitHub:

git clone https://github.com/DandaAkhilReddy/ReddyHedgefund.git && cd ReddyHedgefund
2

Set Up Environment

Install dependencies and configure API keys:

pip install -r requirements.txt && cp .env.example .env

Add your OPENAI_API_KEY and FINANCIAL_DATASETS_API_KEY to .env

3

Run the Hedge Fund

Launch the multi-agent analysis pipeline:

python src/main.py --ticker AAPL --show-reasoning
