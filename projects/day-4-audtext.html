Transform audio files into text with AI-powered summaries. All processing happens locally on your machine - your audio never leaves your computer. Powered by OpenAI's Whisper for transcription and Llama 3.1 via Ollama for intelligent summarization. Zero API costs, forever.

The Problem

Audio transcription services have major drawbacks:

  • Cloud services are expensive per minute
  • Privacy concerns - audio uploaded to servers
  • No control over your sensitive data
  • API costs add up quickly for long files
  • Summaries require separate paid services

The Solution

Audtext brings everything local:

  • 100% local processing - nothing uploaded
  • Free forever - no API keys required
  • Whisper for accurate transcription
  • Llama 3.1 for intelligent AI summaries
  • Export to TXT, SRT, VTT, or JSON

Key Features

100% Local Processing

All transcription and summarization happens on your machine. Your audio never touches the internet.

Whisper Models

Choose from tiny (~150MB) to large models. Balance speed vs accuracy based on your needs.

AI Summarization

Llama 3.1:8b generates intelligent summaries of your transcripts locally in ~30 seconds.

Multi-Format Export

Export transcripts as TXT, SRT subtitles, VTT for web, or structured JSON data.

Real-Time Progress

Track transcription and summarization progress with live status updates in the UI.

Multi-Language

Automatic language detection. Transcribe audio in multiple languages with Whisper.

Quick Start Guide

1

Prerequisites

Make sure you have these installed:

Python 3.11+
Node.js 18+
FFmpeg (for audio processing)
Ollama (for local LLM)
2

Clone & Setup Backend

git clone https://github.com/DandaAkhilReddy/Audtext.git
cd Audtext
python -m venv venv

# Windows:
venv\Scripts\activate

# Mac/Linux:
source venv/bin/activate

pip install -r requirements.txt
3

Setup Frontend

cd frontend
npm install
4

Download AI Model

Download the Llama 3.1 model for summarization (~4.7GB):

ollama pull llama3.1:8b
5

Run the Application

Open 3 terminal windows and run:

# Terminal 1: Start Ollama server
ollama serve

# Terminal 2: Start Backend (from project root)
cd backend
uvicorn main:app --reload --port 8000

# Terminal 3: Start Frontend (from frontend directory)
cd frontend
npm run dev
6

Open in Browser

Access the application at:

http://localhost:5173
